Console Keybindings (as shipped)
--------------------------------
*  F1 = help
*  F2 = status
*  F3 = show channels
*  F4 = show calls
*  F5 = sofia status
*  F6 = reloadxml
*  F7 = console loglevel 0
*  F8 = console loglevel 7
*  F9 = sofia status profile internal
* F10 = sofia profile internal siptrace on
* F11 = sofia profile internal siptrace off
* F12 = version
